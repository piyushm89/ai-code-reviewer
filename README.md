# AI Code Reviewer

AI Code Reviewer is a full-stack web application that leverages Mistral AI to analyze source code and provide intelligent feedback. The platform helps developers improve code quality, identify potential bugs, optimize performance, follow best practices, and enhance maintainability.

## Features

* AI-powered code analysis using Mistral AI
* Syntax-highlighted code editor
* Markdown-formatted review responses
* Performance and security recommendations
* Clean and responsive user interface
* Real-time code review experience

## Tech Stack

### Frontend

* React.js
* Vite
* Prism.js
* Axios

### Backend

* Node.js
* Express.js
* Mistral AI API

## Use Cases

* Review code before deployment
* Learn coding best practices
* Identify performance bottlenecks
* Detect potential security issues
* Improve code readability and maintainability

This project demonstrates the integration of Large Language Models (LLMs) into modern developer tools to automate and streamline the code review process.
